# OTPLESS (REACT + NODEJS)


Complete Setup With FrontEnd & BackEnd


### Clone this repository 
```
https://github.com/ProgrammerShri/otpless.git
```

### Start React Frontend
```
npm install && npm start
```

### Start Node Backend
```
cd backend && npm install && npm start
```
